# pet-manager-python

A Python command-line application to register and filter pets, 
built using Object-Oriented Programming (OOP) principles.

## About
This program allows users to register multiple pets and view them 
either all at once or filtered by animal type. Built as part of 
a lab exercise at Dalarna University, focusing on classes, private 
attributes, and OOP design.

## Features
- Pet class with private attributes (name, animal type, age)
- Getter and setter methods for all attributes
- User input via a counter-controlled while loop
- Stores pet objects in a list
- Display all pets or filter by animal type
- Full input validation throughout

## How to Run
```bash
python animals.py
```

## Built With
- Python 3
- OOP — classes, private attributes, getter/setter methods
- Input validation functions for clean, reusable code
